Comprueba si un grafo es cíclico o no.
Los datos pueden introducirse manualmente como se indica o a través de un archivo casos.txt
formato de casos.txt con ejemplo:
5
4
1 2
3 2
0 2
-1 -1

El primero número (5), indicará el número de casos a evaluar.

Después, se repetirán 5 veces lo siguiente, una para cada caso.
El segundo número(4) indica el número de vértices que habrá, empezando en 0.
Las siguientes líneas introduce aristas, con número del vértice origen seguido del destino.
Para acabar el caso: -1 -1
